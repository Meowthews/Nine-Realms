Release History

Release History

This page tracks released and unreleased changes.

Version 1.3.0 Unreleased

Journal Access And Menu Flow Updates

  • Moved access to the Nine Realms menu into the Journal with a dedicated bottom button, replacing the old standalone cultivation menu action flow.
  • Retired the legacy menu ability path and added cleanup so older menu ability data does not linger on the player.
  • Hardened cultivation ability so it is no longer used by auto combat.
  • Refreshed the cultivation menu layout into a larger tabbed panel and shifted wiki access toward the external Nine Realms web documentation.
  • Adjusted cultivation costs from the older base 4 scaling model to the current stepped formal-path costs shown in the live release branch.
Version 1.2.0 Unreleased

Shared Immortal Expansion

  • Expanded the higher shared immortal realms from five to nine, raising the current shared cap to 18000.
  • Shared immortal realms now provide broad cultivation gains, breakthrough rewards, and passive bonuses for formal Body and Qi cultivators.
  • The cultivation menu now shows current stamina and MP cost per cultivation tick.
  • Shared immortal realms now have slightly higher stat gain chance and stronger shared HP, MP, and stamina breakthrough rewards.
Version 1.1.1 Unreleased

Stability And UI Corrections

  • Fixed a feat menu stability issue caused by cultivation path preview entries mutating player feat data.
  • Hardened Cultivation Initiate ability sync so Cultivate and the Cultivation Menu are granted more safely after purchase.
  • Corrected Rogue Cultivator menu progress display to show the current shared cap instead of the first formal breakthrough gate.
Version 1.1.0 Released

Localization And Rogue Growth

  • Added Chinese and Japanese localization.
  • Reduced Body Cultivator and Qi Practitioner feat costs from 5 to 2.
  • Removed the 999 progress cap for Rogue Cultivators. Rogue Cultivators can now progress up to the same current cap as formal paths, but still cannot break through.
Version 1.0.0 Released

Initial Release

  • Initial release of Nine Realms: Path of the Dao.
  • Added the Cultivation Initiate feat and the Cultivate long action.
  • Added hidden cultivation progress, realm thresholds, stages, and breakthrough gates.
  • Added Body Cultivator, Qi Practitioner, and Rogue Cultivator support.
  • Added passive realm bonuses, breakthrough rewards, the cultivation menu, the in-game Nine Realms Wiki, and the changelog window.
